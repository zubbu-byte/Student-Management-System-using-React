# Student-Management-System-using-React
Student Management System  A full-stack MERN (MongoDB, Express.js, React, Node.js) application that provides a secure platform for managing student records. The application includes JWT-based authentication to ensure only authorized users can access and manage student data.
# Student Management System

A full-stack MERN application for managing students with JWT authentication.

## Features

- User Registration
- User Login
- JWT Authentication
- Protected Routes
- Add Student
- View Students
- Delete Student
- MongoDB Database

## Tech Stack

- React
- Vite
- Express.js
- MongoDB
- Mongoose
- JWT
- Axios
- Node.js

## Installation

### Backend

```bash
cd backend
npm install
npm start
```

### Frontend

```bash
cd frontend
npm install
npm run dev
```

## Environment Variables

Create a `.env` file inside the backend:

```env
MONGO_URL=your_mongodb_url
JWT_SECRET=your_secret_key
PORT=8000
```
